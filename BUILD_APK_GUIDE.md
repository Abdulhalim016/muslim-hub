# Muslim Hub - APK বিল্ড গাইড

## Expo EAS থেকে APK বিল্ড করার ধাপ

আপনার Muslim Hub অ্যাপ এখন Expo EAS এ বিল্ড করার জন্য প্রস্তুত। নিচের ধাপগুলো অনুসরণ করুন:

### ধাপ ১: Expo ওয়েবসাইটে লগইন করুন

1. আপনার মোবাইল বা কম্পিউটারে ব্রাউজার খুলুন
2. এই লিংকে যান: https://expo.dev
3. আপনার অ্যাকাউন্টে লগইন করুন
   - ইউজারনেম: `Abdulhalim016`
   - পাসওয়ার্ড: আপনার পাসওয়ার্ড

### ধাপ ২: নতুন প্রজেক্ট তৈরি করুন

1. Expo ড্যাশবোর্ডে "Create a new project" ক্লিক করুন
2. প্রজেক্ট নাম: `muslim-hub`
3. "Create" বাটন ক্লিক করুন

### ধাপ ৩: প্রজেক্ট কোড আপলোড করুন

আপনার কাছে দুটি অপশন আছে:

#### অপশন A: GitHub থেকে (সুপারিশকৃত)

1. GitHub এ যান: https://github.com
2. নতুন Repository তৈরি করুন:
   - Repository name: `muslim-hub`
   - Description: "Muslim Hub - Islamic App"
   - Public বেছে নিন
   - "Create repository" ক্লিক করুন

3. আপনার প্রজেক্ট ফাইল GitHub এ আপলোড করুন:
   - `muslim-hub.zip` এক্সট্র্যাক্ট করুন
   - সব ফাইল GitHub এ আপলোড করুন

4. Expo এ GitHub Repository কানেক্ট করুন:
   - Expo ড্যাশবোর্ডে আপনার প্রজেক্ট খুলুন
   - "Connect repository" ক্লিক করুন
   - আপনার GitHub Repository সিলেক্ট করুন

#### অপশন B: সরাসরি Expo CLI (কম্পিউটার প্রয়োজন)

কম্পিউটারে এই কমান্ড চালান:
```bash
cd muslim-hub
npx eas build --platform android
```

### ধাপ ৪: APK বিল্ড করুন

1. Expo ড্যাশবোর্ডে আপনার প্রজেক্ট খুলুন
2. "Build" ট্যাব ক্লিক করুন
3. "Create new build" ক্লিক করুন
4. প্ল্যাটফর্ম: `Android` বেছে নিন
5. বিল্ড টাইপ: `APK` বেছে নিন (অথবা `AAB` যদি Google Play এ আপলোড করতে চান)
6. "Build" বাটন ক্লিক করুন

### ধাপ ৫: APK ডাউনলোড করুন

1. বিল্ড প্রসেস শেষ হওয়ার জন্য অপেক্ষা করুন (১০-২০ মিনিট)
2. বিল্ড সম্পন্ন হলে "Download" বাটন দেখা যাবে
3. APK ফাইল ডাউনলোড করুন

### ধাপ ৬: আপনার মোবাইলে ইনস্টল করুন

1. APK ফাইল আপনার মোবাইলে ট্রান্সফার করুন
2. ফাইল ম্যানেজার খুলুন
3. APK ফাইল খুঁজুন এবং ট্যাপ করুন
4. "Install" বাটন ক্লিক করুন
5. ইনস্টলেশন সম্পন্ন হলে "Open" ক্লিক করুন
